Skin Cancer Detection Using Machine Learning

This project was centered around using deep learning to help identify skin cancer through dermoscopic image analysis. Using the ISIC2018 dataset, which consists of thousands of labeled skin lesion images, I trained a Convolutional Neural Network (CNN) model capable of classifying images as benign or malignant. I also integrated ESRGAN-based super-resolution techniques to improve image clarity and model precision. My primary contribution included designing the data preprocessing pipeline, training the CNN, evaluating the model using various metrics, and interpreting the results to ensure a clinically meaningful output. This project aims to support dermatologists by providing a quick, accurate pre-screening tool.

Online Movie Ticket Booking System

This project involved creating a web-based application that allows users to search, book, and manage movie tickets from the comfort of their homes. The system featured user authentication, dynamic seat selection, movie filtering, and a real-time booking database. I developed the front-end using HTML, CSS, and JavaScript, and the back-end using PHP and MySQL. My major responsibilities included building the UI, developing the ticketing logic, and managing the database schema. This platform aimed to replicate the functionality of commercial ticketing apps with a focus on simplicity and user convenience. It significantly reduced manual effort in ticket management and improved accessibility.

RFID Smart Lock System

The RFID Smart Lock System was designed to enhance physical security using RFID-based authentication. Using an Arduino microcontroller, RFID reader, servo motor, and buzzer, I built a smart locking mechanism that opens upon detecting an authorized RFID tag. My contribution involved wiring the components, coding the Arduino logic to handle verification and motor control, and testing the system under different conditions to ensure reliability. I also added an alert system for unauthorized access attempts. The project presents a scalable, low-cost alternative for traditional locks, aimed at increasing security in homes, hostels, or offices with digital key access.

Deepfake Detection

This project involved contributing to a research paper on deepfake detection, addressing the growing challenge of AI-generated media manipulation. I conducted extensive web-based research to identify state-of-the-art deepfake detection techniques, focusing on methods like Convolutional Neural Networks (CNNs) and Graph Neural Networks (GNNs), as well as benchmark datasets such as FaceForensics++ and the Deepfake Detection Challenge. My role included gathering and analyzing over 50 recent studies, synthesizing findings on detection accuracy and computational efficiency, and assisting in drafting sections on methodology and future research directions. The paper aims to provide a comprehensive review to guide future advancements in countering deepfake-related misinformation.
